# PR: Add TaskRelay to Awesome Intercom

## App Name
**TaskRelay** — P2P Task Delegation & Confirmation Board

## Description
A two-role peer-to-peer task board built on Intercom. An admin peer posts tasks over an Intercom sidechannel; worker peers claim and complete tasks in real time. No server, no database — pure P2P via Pear runtime and Hyperswarm.
